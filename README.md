# Atom2Universe

Atom2Universe is an ever-growing Android app made of many different modules. At its core is an **atom clicker** — you start with a single atom and collect more to gradually fill your universe. Around that core, the app keeps expanding with new modules constantly being added.

> **Note:** The source code is open-source, but the **Atom2Universe brand** (name, logo, icon, store assets) is protected. See [`docs/TRADEMARK.md`](./docs/TRADEMARK.md)

---

## Modules

| Module | Description |
|--------|-------------|
| Atom clicker | Idle/clicker game — collect atoms, fill your universe, unlock gacha rewards |
| Music player | Local library, playlists, equalizer DSP, Google Drive sync, lyrics |
| MIDI player | Dual engine (Sonivox EAS + custom SF2 synthesizer), SoundFont support |
| Radio | Internet radio streaming with recording |
| Audio editor | Recording, editing via FFmpeg |
| SF2 creator | Build your own SoundFonts |
| Games | Chess, sudoku, blackjack, 2048, solitaire, Stars War, and more |
| Creative tools | Pixel art, canvas, notes |
| Book reader | EPUB reader with library management |
| Comics reader | CBZ/CBR reader with grid and tree view |

---

## Status

- Platform: Android (minSdk 26)
- Development: active, constantly evolving
- Architecture: modular (native C + Java + Kotlin)
- Localization: 14 languages — English, French, German, Spanish, Italian, Greek, Portuguese, Dutch, Romanian, Russian, Turkish, Ukrainian, Polish, Indonesian

---

## Build

1. Open the project in Android Studio
2. Sync Gradle
3. Run on a physical device or emulator (arm64-v8a)
