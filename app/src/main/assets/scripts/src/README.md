# Scripts modulaires (work in progress)

Ce dossier regroupe progressivement les morceaux extraits de `scripts/app.js`.
L'objectif est d'obtenir un `app.js` mince qui orchestre l'initialisation.

## Structure cible (premiers blocs)
- `core/` : logique de base (flags globaux, état global, boucle principale).

## Notes pour contribuer
- Chaque module doit expliquer son rôle en en-tête.
- Les scripts sont chargés par `index.html` avant `app.js`.
- Conserver un ton pédagogique dans les commentaires si besoin.
