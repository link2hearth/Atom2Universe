# Atom2Universe — Trademark & Brand Guidelines

The Atom2Universe source code may be made available under an open-source license (e.g., Apache-2.0),
but the **Atom2Universe** name and brand assets are **not** granted under that code license.

## What is covered (“Atom2Universe Marks”)
The following are considered **Atom2Universe Marks**:
- The name **Atom2Universe**
- The Atom2Universe logo(s), icon(s), and visual identity
- App listing assets (screenshots, promotional images, descriptions)
- Any distinctive branding elements (colors/layouts used as identifiers), when they create confusion

Unless explicitly stated otherwise, these items are **not licensed for reuse**.

## Allowed uses (no permission needed)
You may:
- Refer to **Atom2Universe** to discuss the project (blog posts, reviews, tutorials, comparisons)
- Link to the official repository and releases
- Fork the code for personal use, experiments, or contributions (pull requests)
- Redistribute **unmodified** source code copies that keep the original notices
- State that your project is “based on Atom2Universe” **as long as it’s clear it is unofficial**
  and you do not use the Atom2Universe Marks as your product identity

## Not allowed (without written permission)
You may **not**:
- Publish an app (on Google Play or elsewhere) using the **Atom2Universe** name, icon, or branding
- Present a fork/clone as the official Atom2Universe app, or in a way that can confuse users
- Use Atom2Universe Marks in marketing, metadata, package name, listing text, or screenshots
- Use “Atom2Universe” in a product name (e.g., “Atom2Universe Pro”, “Atom2Universe Lite”) for a fork
- Use the Atom2Universe icon/logo (or confusingly similar ones) for your own distribution

## Fork naming rule (recommended)
If you publish a fork, pick a new name and branding.
Recommended wording:
- “Unofficial fork of Atom2Universe” (in README only, not as the app/store title)
- “Based on the Atom2Universe project” (attribution without branding reuse)

## Reporting and permission requests
If you want to use Atom2Universe branding, or if you see misuse, contact:
- Contact@Atom2Universe.app

## Notes
- This document is intended to reduce user confusion and prevent impersonation.
- This is not legal advice.
